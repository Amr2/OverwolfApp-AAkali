Spell Tracker

Spell Tracker is an easy to use MANUALLY summoner spell cooldown timer overlay.

Spell Tracker is designed to help players track the enemy's summoner spells

Our main fnction is the overlay but we also have a desktop feature where the user is able to generate an MMR graph with an estimated current MMR value for the player along with a "top MMR" and "minimum MMR" portion

a Tutorial for desktop part .
